Desarrollo de Aplicaciones Web

Formación Profesional Superior

El trabajo final de primero de DAW consistió en el desarrollo de un sitio web de escape rooms online.

Para entregar una aplicación final, debíamos cumplir con los requerimientos de un cliente:

  • Análisis funcional
  • Mock-up de la web
  • Despliegue de base de datos SQL
  • Programación del modelo en Java
  • Maquetación responsive en CSS y HTML
  • Servidor en máquina virtual
  • Web multilingüe

Requerimientos de la aplicación

Pincha para ver el documento

Tuvimos una entrevista con un cliente, el cual nos explicó sus necesidades. Algunas eran, por ejemplo, de que la aplicación debería ser multilingue y que se debían poder hacer reservas para salas físicas.

A raiz de todo esto, para empezar el proyecto creamos un enunciado el cual debíamos cumplir hasta el final (el análisis de la aplicación, con su diagrama de clases y casos de uso).
